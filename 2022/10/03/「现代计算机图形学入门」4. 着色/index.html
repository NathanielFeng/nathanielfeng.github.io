「现代计算机图形学入门」着色

解决了如何把模型投影在屏幕上、光栅化后，需要解决的问题是：每个像素应该是什么颜色？ 即着色。

本课中，着色定义：对不同物体应用不同材质的过程。

1. Blinn-Phong 反射模型

Blinn-Phong 反射模型是一个种经验模型。

它把光源分成 3 类：高光（Specular highlights）、漫反射（Diffuse reflection）、环境光（Ambient lighting）

着色具有局部性，因为不考虑其他任何物体的存在，只考虑自己！所以不考虑阴影！

1.1 漫反射

与观测方向无关，其中，\(L_d\) 是漫反射光，\(k_d\) 是慢反射系数，\(I/r^2\) 是到达 shading point（着色点）的能量，\(max(0, n·l)\) 是着色点接收到的能量。 \[ L_d = k_d(I/r^2)max(0, n·l) \]

1.2 高光

为了加速，引入了半程向量 \(h = bisector(v, l) = \dfrac{v+l}{||v+l||}\)，

其中，\(L_s\) 是高光，\(k_s\) 是镜面反射系数，表示亮度，\(p\) 是衰减指数。 \[ L_s = k_s(I/r^2)max(0, n·h)^p \]

1.3 环境光

常数项，其实不准确，准确的涉及到 全局光照 的内容。 \[ L_a = k_aI_a \]

1.4 Blinn-Phong 光照模型

\[ L = L_a + L_d + L_s \]

2. 着色频率

着色频率，即把着色应用在哪些点上。

Flat Shading

  • 对每个三角形面进行一次着色，对于光滑表面不好。

Gouraud Shading

  • 对每个顶点进行着色，然后对颜色进行插值。

  • 顶点法线：共顶点面的法线的平均。

Phong Shading

  • 对每个像素进行着色。

2.2 纹理映射

模型中每一个三角形顶点都被赋予一个纹理坐标 \((u,v)\)。

通常对于一张纹理来说，\(u,v\) 都在 \([0, 1]\) 之内。

着色与纹理的关系：纹理是定义着色的时候，各个不同点的属性。

使用重心坐标进行简单的纹理映射：漫反射

2.3 重心坐标

重心坐标是为了对三角形内部进行插值，希望能够获得一个顶点到另一个顶点平滑的过渡。

哪里可以用到插值 ？位置、纹理坐标、颜色、法线、深度、材质属性。

注意：重心坐标在投影下不能保证不变。所以如果我们想插值三维空间中的属性，比如深度，我们应该插值三维空间中的坐标，而不是等到投影后做。

3.1 纹理放大(Texture Magnification)

双线性插值（Bilinear）

其中，黑色的点是 texel，红色的点是 pixel。

双立方插值（Bicubic）

3.2 纹理缩小(Minification)

问题：近处会出现锯齿、远处会出现摩尔纹。

原因是：当纹理特别大的时候，像素内部就会包括特别大的纹理，这一大块纹理的纹素是不同的（即一直在变化），但是只用了一个像素去采样。所以发生了走样。

超采样有用，但是开销太大！

Mipmap：允许快、近似、正方形的范围查询

仅仅多引入了 \(1/3\) 的存储空间！

如何计算Mipmap Level D ?

该查找过程是一个从XY空间到UV空间的一个映射。我们可以通过分别求x和y偏导数来求屏幕单个像素宽度纹理坐标的变化率。举个例子，屏幕上某像素区域，对应到实际的纹理中可能是一个长方形的区域。x轴方向实际texel覆盖率为1，y轴的实际texel覆盖率为4。我们可以用ddx和ddy分别来求这个两个方向上的覆盖率，然后取较大的覆盖率作为该像素覆盖纹理正方形的边长。

如何确定 Mipmap 等级 \(D\) 呢？

如果 \(L=1\)，则覆盖一个纹素的面积，我们查询原始纹理图即可。

如果 \(L=2\)，则覆盖四个纹素的面积，我们得查询一级纹理图才行，因为其边长缩小了一半，把四个纹素平均成了一个。

所以可以发现，\(D = log_2L\)。

3.3 Mipmap可视化后发现的问题

我们把图片中用到的层数 \(D\) 可视化后发现，\(D\) 层数是不连续的，这样子层与层之间可能会产生缝。

三线性插值（Trilinear Interpolation）

在 Level D 和 Level D+1 间再次做一次插值。

3.4 Mipmap局限性

Overblur ？ 模糊的过分了！

各项异性过滤（Anisotropic Filtering）

允许我们对一个矩形的区域做一个范围查询。但是对于斜着的区域不行！

4. 纹理的应用

在现代 GPUs 中，纹理 = 内存 + 范围查询（过滤）

纹理可以理解为一块数据，可以做不同的查询。

Environment Map

经典模型：犹他茶壶、斯坦福兔子/龙、康奈尔盒子

由于用球（Spherical Map）记录不同方向的光照信息（环境光）的贴图展开成矩形后，上下两极会发生扭曲现象（类比世界地图），于是用球的包围盒（Cubic Map）来记录贴图。

4.1 纹理还可以影响shading

纹理不仅只代表颜色 \(K_d\)，还可以存储其他信息。

凹凸贴图：通过修改模型表面的法线让模型看起来好像是“凹凸不平”的样子，增加细节层次感，达到高模的效果。但是可以从模型的轮廓边缘处看出“破绽”。通常通过存储高度值实现。

把任何一个像素的法线做一个扰动：通过邻近位置的高度差重新计算法线。

此时，纹理记录的是 任意一个像素“高度的移动”。

法线贴图：直接存储法线信息。
